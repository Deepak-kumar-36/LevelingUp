# Leveling UP

A Personal Life Operating System designed to help you track habits, manage goals, log your finances, track health metrics, and take notes, all inside a beautifully crafted, brutalist UI.

## Features

- **Habit Tracking**: Define your daily quests and track completions.
- **Weekly Goals**: Set dynamic numeric goals every week.
- **Health & Fitness**: Log daily calories, protein, and track your weight over time.
- **Finances**: Track monthly budgets and log expenses by category.
- **Shop**: Set custom rewards you can buy using coins earned from habits.
- **Notes**: Organize your ideas directly within the app.
- **Local Persistence**: All your data stays on your device using IndexedDB (Dexie).

## Getting Started

1. `npm install`
2. `npm run dev` (Runs locally)
3. Connect your Android or iOS device to the local server or build natively via Capacitor.
